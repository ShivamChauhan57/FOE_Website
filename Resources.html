    DFX Lab Website


    Advanced Visualization Center (AVC)

    Locations:
    • 3D-Printing Lab: CMC 153
    • Advanced Visualization Auditorium: CMC 147
    • Vizualization Computer Lab: SCA 222
    • Shimberg Library 3D Printing Lab: Shimber Library (USF Health Campus)
    AVC offers 3D printing and visualization to students and faculty for assistance in academic and research projects.

Software

    Arduino IDE

    Arduino Integrated Development Environment (IDE) is an open source software used to write code that can be uploaded onto any Adruino board. For more information and to download, visit:
    Arduino IDE Download


    TinkerCAD

    TinkerCAD is an online computer-aided design (CAD) program used to assist in the fabrication of 3D models and circuits. The program is easy to use for those with no previous CAD experience.
    TinkerCAD


    Fusion 360

    Fusion 360 is a software platform used for product design and manufacturing. It includes CAD, CAM, CAE, and PCB softwares. For more information and to download, visit:
    Fusion 360 Download
